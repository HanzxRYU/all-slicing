Kumpulan Tugas Slicing UI (Flutter)

Repositori ini adalah koleksi pribadi saya yang berisi proyek-proyek kecil dan tugas-tugas untuk melatih kemampuan slicing UI. Slicing adalah proses mengkonversi desain statis (seperti dari Figma, Sketch, atau gambar) menjadi kode antarmuka pengguna (UI) yang fungsional dan responsif.

Semua proyek di sini dibuat menggunakan Flutter dan Dart.
